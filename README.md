# GeoPreview — Open GeoJSON, Shapefile, GPKG on Mac

**GeoPreview** is a macOS app that lets you instantly preview geospatial files directly in Finder.

No need to open heavy GIS software like QGIS just to check what's inside a file.

## Open GIS files on Mac instantly

Just press **Space** in Finder and instantly preview:

- GeoJSON  
- Shapefile (.shp)  
- GeoPackage (.gpkg)  
- KML / KMZ  
- GPX  
- LAS / LAZ  
- DXF / DWG  
- OSM (XML / PBF)  
- and many more  

GeoPreview shows a real map preview, attribute table, and spatial metadata — all in one place.

---

## Why GeoPreview?

Opening GIS files on macOS is still painful:

- QGIS takes time to launch  
- Many formats don’t have native preview  
- You often just want to quickly inspect data  

GeoPreview solves this by integrating directly into Finder.

Instead of:

> open QGIS → load layer → wait → zoom → inspect → close  

You just:

> press Space → see everything instantly  

---

## Key features

- Preview GIS files directly in Finder  
- Real map rendering (not just icons or metadata)  
- Attribute table view  
- Geometry inspection  
- Spatial metadata (CRS, bounds, layers)  
- Multi-layer dataset support  
- Dataset folder preview (.gdb, shapefile folders)  
- Quick export to other formats  
- Works fully offline — no uploads, no tracking  

GeoPreview is built for developers, GIS analysts, and anyone working with spatial data.

---

## Supported formats

GeoPreview supports most common geospatial, CAD, and point cloud formats:

- GeoJSON / GeoJSONSeq  
- TopoJSON  
- GeoPackage (GPKG)  
- FlatGeobuf (FGB)  
- Shapefile (SHP)  
- KML / KMZ  
- GPX  
- GML  
- MapML  
- DXF / DGN / DWG  
- OSM (XML / PBF)  
- MVT / PMTiles  
- GeoParquet / Parquet  
- LAS / LAZ  

Also supports dataset folders:

- ESRI File Geodatabase (.gdb)  
- folders with shapefiles or parquet layers
